# Agent Instructions

This project is a mobile web app for recording date-based events.

Please follow the project plan in [PLAN.md](PLAN.md).

## Technical Stack
- Vanilla JS
- CSS
- LocalStorage for persistence
- No outside dependencies

## Key Features
- Chronological display of entries.
- Efficient input with custom date picker and auto-expanding text area.
- Entry duplication (date only).
- Export to text.
- Light, Dark, and Sepia themes.
